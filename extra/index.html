This wizard will configure your computer for secure network access.
To ensure your security, AirPennNet utilizes a secure authentication mechanism known as 802.1X. This security mechanism protects your PennKey user name and password. In a wireless environment, it also protects your data with network encryption. To utilize this secure network, your network connection requires specific settings. This wizard will ensure that your machine is properly configured.

XpressConnect will attempt to load automatically using a Java Applet or ActiveX. If you prefer, you may load XpressConnect manually.

During the configuration process Information Systems and Computing (ISC) will collect certain data regarding the device that you are using to connect to AirPennNet. For more information please read our privacy statement.
